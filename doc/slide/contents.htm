Configuration file

Line format: set :: value
customTabs
Custom tabstops (1,10,55 character) - default: 1 (on)
editorName
External editor - default: gvim
editorParam
Editor commandline parameters (has to be started in foreground) - default: -f
snobol4prog
Snobol4 executable - default: snobol4-1.1\bin\ms-snobol4.exe
snobol4option
Snobol4 options - default: -b (no banner)
snobol4param
Snobol4 program parameters - default: none
inputfile
Name of the input file - default: snobol4.in
outputfile
Name of the output file - default: snobol4.out
errfile
Name of the error file - default: snobol4.err
myFont
Interface font - default: Courier 10
encoding
Encoding - default: utf-8
browser
Path to the browser - default: C:\program files\internet explorer\iexplore.exe
geom
Geometry on startup - default: 800x600
syntax
Syntax highlighting - default: 1 (on)
useUTFlibrary
Use UTF.sno when running programs? - default: 0 (no)
